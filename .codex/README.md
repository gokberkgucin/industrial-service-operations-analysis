# Repo-Specific Codex Notes

Bu klasör, repo içindeki Codex çalışma bağlamını belgelemek için kullanılır. Bağlayıcı kuralların ana kaynağı kökteki `AGENTS.md` dosyasıdır.

## Çalışma Amacı

- Hedef, staj defterini GitHub Pages üzerinde sayfa sadakati yüksek biçimde yayımlamaktır.
- DOCX kaynak dosyası ve sayfa sırası esas alınır.
- Mevcut portföy metinleri yardımcı bağlam olabilir, fakat canonical source değildir.

## Yerel Üretim İlkesi

- Üretim hattı internet bağımlılığı olmadan çalışmalıdır.
- LibreOffice, Pandoc, ImageMagick ve Python tabanlı araçlar yerel dönüşüm için tercih edilir.
- Ham ve ara çıktılar `_work/` veya `work/` altında tutulur.

## Faz Sonu Alışkanlığı

Her faz sonunda kısa summary, verification ve değişen dosya listesi bırakılır.
